# Sales-Dashboard
📊 Sales Dashboard (Excel Project)

🚀 Overview

This project represents my first hands-on experience working with Excel.
I built an interactive Sales Dashboard to analyze and visualize sales data in a clear and structured way.

📌 Key Features

* Interactive dashboard design
* Data visualization for better insights
* Sales analysis by time, country, and product
* Summary metrics (Total Sales, Total Orders, Avg Delivery Time)

🛠️ Skills & Tools

* Microsoft Excel
* Pivot Tables
* Data Cleaning
* Data Visualization

📈 What I Learned

Through this project, I learned how to:

* Use Pivot Tables to organize and analyze data
* Transform raw data into meaningful insights
* Build simple and interactive dashboards
